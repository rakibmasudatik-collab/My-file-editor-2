# Telegram Premium File Editor Bot 🚀

A high-performance mini VS Code inside Telegram built with Python 3.12+ and python-telegram-bot v21.

## Features
- Create, Edit, Rename, Delete, Duplicate files
- Auto-Save with Dynamic Version History & Restore
- Supports text files (py, js, html, css, json, etc.)
- Full ZIP Project Creation, Extraction, and Downloading
- Secure Password-Protected Admin Panel
- Asynchronous SQLite Database

## Local Setup
1. Clone the repo
2. Create python environment: `python -m venv venv`
3. Activate environment: `source venv/bin/activate` (Linux/macOS) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Copy `.env.example` to `.env` and enter your BOT_TOKEN
6. Run: `python main.py`
